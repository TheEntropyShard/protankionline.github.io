Добро пожаловать на страницу тестовых серверов «ProTanki»!

Большую часть времени серверы работают в режиме закрытого тестирования: доступ к ним есть только у тестеров, круг которых заранее определён разработчиками игры. Вход на закрытый тестовый сервер возможен только с инвайт-кодом. Помните, что инвайт-коды не выдаются игрокам, и их невозможно получить на сторонних сайтах.

Иногда, при необходимости проведения нагрузочных тестов, серверы открыты для всех игроков. В этом случае инвайт-код для входа не требуется.

Помните, что звание, очки и кристаллы, заработанные на тестовых серверах, никогда не попадут на основные.

Welcome to the ProTanki test servers page!

Most of the time, these servers work in closed testing mode. This means they can only be accessed by a group of testers which the developers have determined in advance. Entering a closed test server is only possible with an invite code. Please note that invite codes are not issued to players, and it is impossible to get them on any sites.

Sometimes, when there is need for a mass test, servers are opened up to all players. In such cases, invite codes are not requested.

Remember that rank, score and crystals earned on test servers will never appear on the main servers.
